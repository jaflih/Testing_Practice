# Testing Practice

> Learning objectives

- Write unit tests for a JavaScript app.
- Use AAA pattern for unit tests.
- Explain why testing code is important.

## Built With

- JavaScript
- [Jest](https://jestjs.io/)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Install

```
npm install
```

### Run tests

```
npm run test
```
